# NYC 311 Customer Service Analysis

## 📌 Project Overview
This project performs a comprehensive data analysis on New York City's 311 service requests. The goal is to evaluate customer service efficiency, analyze response and resolution times across different boroughs, and identify the most frequent complaint categories. By analyzing this data, the project aims to uncover actionable insights to optimize resource allocation and improve city-wide citizen satisfaction.
